# Contributing to Paperback

Thank you for your interest in contributing to Paperback! We welcome bug reports, feature requests, documentation improvements, and pull requests.

## How to Contribute

1. **Report Bugs**: If you find a bug, please open an issue describing the problem, how to reproduce it, and your environment (browser/OS).
2. **Suggest Features**: Have an idea to make Paperback better? Open a feature request issue!
3. **Submit Pull Requests**:
   - Fork the repository.
   - Create a branch for your fix/feature.
   - Commit your changes with clear messages.
   - Push to your fork and submit a Pull Request.

## Code of Conduct

Please be respectful and collaborative when engaging with other contributors.
